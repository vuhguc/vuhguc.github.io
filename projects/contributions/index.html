Career Experiences

Software Developer

The Innovative Clinical Data Capture and Use Lab

This is my current co-op rotation. I have worked here part-time since August 2019 and registered as a co-op rotation in Fall 2020. iCDCU is a research lab found by Dr. Danny Wu focusing on applications of information technologies in biomedical fields. I have worked on many different projects here; the most recent one was using Elasticsearch and Learning to Rank to retrieve medical text. I have learned many new concepts and technologies in Natural Language Processing, most notably Python libraries NLTK and scikit-learn. One non-technical concept I have learned is working in an agile style (dividing a project into small development stages).

August 2019 - Present

Research Assistant

UC Professor John Franco’s Lab

This was my first co-op rotation. My main project there was developing a cyber-security wargame in C. The game included a monitor that worked as a server; players were clients connecting to the server and playing the game according to a set of rules. I developed the monitor in C and a simple client user interface in Java Swing. The most important technical concepts I learned were multithreading and socket programming in C. I also learned how to read and learn new concepts from research paper.

August 2018 - December 2018
